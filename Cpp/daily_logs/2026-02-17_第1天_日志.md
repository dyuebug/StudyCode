## 📖 学到的知识点
1. **C++程序基本结构**
   - #include预处理指令的作用
   - using namespace std的含义
   - main()函数是程序入口
   - return 0表示程序正常结束

2. **数据类型**
   - int（整数）、float（单精度浮点数）、double（双精度浮点数）
   - char（字符）、bool（布尔值）、string（字符串）
   - 各类型的内存占用和取值范围

3. **输入输出**
   - cout << 用于输出
   - cin >> 用于输入（遇到空格停止）
   - getline(cin, str) 用于读取整行（包含空格）
   - endl 换行并刷新缓冲区

4. **格式化输出**
   - #include <iomanip> 提供格式化工具
   - setprecision() 设置小数精度
   - fixed 固定小数点格式
   - setw() 设置字段宽度

5. **编译和运行**
   - 编译命令：g++ -o 输出文件名 源文件.cpp
   - 运行命令：./输出文件名（Linux/Mac）或 输出文件名.exe（Windows）
   - system("chcp 65001") 解决中文乱码问题

## ❓ 遇到的问题与解决
**问题1**：cin读取字符串时遇到空格就停止
**解决**：使用getline(cin, 变量名)读取整行，包括空格

**问题2**：中文输出乱码
**解决**：在程序开头添加system("chcp 65001")设置UTF-8编码

**问题3**：cin >>后使用getline()时跳过输入
**解决**：在cin >>后添加cin.ignore()清除缓冲区的换行符
