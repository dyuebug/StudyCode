## ❓ 遇到的问题与解决

### 问题1：编译错误 - 重复的main函数

**现象**：
```
error: redefinition of 'int main()'
35_file_io_exercise.cpp:513: note: 'int main()' previously defined here
```

**原因**：
- 文件中存在两个main函数（311行和527行）
- 注释块后有重复的代码段（508-635行）

**解决方案**：
```bash
# 备份原文件
cp 35_file_io_exercise.cpp 35_file_io_exercise.cpp.bak

# 删除重复的509-635行
sed -i '509,635d' 35_file_io_exercise.cpp
```

**教训**：
- 复制粘贴代码时要仔细检查
- 一个程序只能有一个main函数
- 注释块结束后不应有代码暴露在全局
