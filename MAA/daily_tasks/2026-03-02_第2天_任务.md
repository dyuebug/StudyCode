## 📌 学习重点

### C++ 核心概念

**条件判断的本质**：
- 让程序具备"决策能力"
- 根据不同条件执行不同代码
- 这是程序"智能"的基础

**if-else vs switch-case**：
- if-else：适合范围判断（如：分数 >= 90）
- switch-case：适合离散值判断（如：选项 1、2、3）

**逻辑运算符**：
- && (与)：所有条件都为真
- || (或)：至少一个条件为真
- ! (非)：取反

### MAA 源码关注点

**typedef 的工程意义**：
```cpp
typedef unsigned char AsstBool;
```
- 为什么不用 C++ 的 bool？
- 提示：跨语言调用时，bool 的大小可能不一致
- AsstBool 明确是 1 字节，保证兼容性

**函数命名规范**：
- 所有函数都以 `Asst` 开头
- 这是 C 语言中避免命名冲突的常见做法
- C++ 有 namespace，但 C API 需要用前缀

**条件编译**：
```cpp
#ifdef _WIN32
    // Windows 特定代码
#else
    // 其他平台代码
#endif
```
- 让同一份代码在不同平台编译出不同结果
- MAA 需要支持 Windows、Linux、macOS
