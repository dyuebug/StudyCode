## 📖 学到的知识点

### 集合（Set）核心知识
1. **集合的概念**：无序、唯一、可变的数据结构
2. **集合的创建**：
   - 花括号：`{1, 2, 3}`
   - set()函数：`set([1, 2, 3])`
   - 空集合：`set()`（注意：`{}`是空字典）
3. **集合的特点**：
   - 无序：不能用索引访问
   - 唯一：自动去重
   - 可变：可以添加、删除元素
4. **集合的基本操作**：
   - 添加：`add()`、`update()`
   - 删除：`remove()`、`discard()`、`pop()`、`clear()`
   - 成员测试：`in`、`not in`
5. **集合的数学运算**：
   - 交集：`&` 或 `intersection()`
   - 并集：`|` 或 `union()`
   - 差集：`-` 或 `difference()`
   - 对称差集：`^` 或 `symmetric_difference()`
6. **集合的使用场景**：
   - 去重：`list(set(列表))`
   - 成员测试：判断元素是否存在
   - 集合运算：找共同元素、不同元素

### 条件判断（if-elif-else）核心知识
1. **条件判断的概念**：让程序根据不同情况执行不同代码
2. **if语句的基本结构**：
   ```python
   if 条件:
       代码块
   ```
3. **if-else双分支结构**：
   ```python
   if 条件:
       代码块1
   else:
       代码块2
   ```
4. **if-elif-else多分支结构**：
   ```python
   if 条件1:
       代码块1
   elif 条件2:
       代码块2
   else:
       代码块3
   ```
5. **嵌套if语句**：if语句内部可以再嵌套if语句
6. **比较运算符**：`==`、`!=`、`>`、`<`、`>=`、`<=`
7. **逻辑运算符**：
   - `and`：与（两个条件都成立）
   - `or`：或（至少一个条件成立）
   - `not`：非（取反）
   - 优先级：`not` > `and` > `or`
8. **条件表达式（三元运算符）**：
   ```python
   结果 = 值1 if 条件 else 值2
   ```
9. **常见陷阱**：
   - 使用`=`而不是`==`
   - 忘记冒号`:`
   - 缩进错误
   - 逻辑运算符的短路特性
