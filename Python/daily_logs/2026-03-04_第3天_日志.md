## 📖 学到的知识点

### 核心概念（15+个）

**列表基础（8个）**：
1. **列表的概念**：有序、可变、可重复的数据集合
2. **列表创建**：[]、list()、列表推导式
3. **列表索引**：正向索引从0开始，反向索引从-1开始（与字符串相同）
4. **列表切片**：[start:end:step]，左闭右开（与字符串相同）
5. **列表可变性**：可以修改元素，与字符串不可变性形成对比
6. **append()**：在列表末尾添加元素
7. **insert()**：在指定位置插入元素
8. **remove()/pop()/del**：删除元素的三种方式

**列表进阶（7个）**：
9. **sort()和sorted()的区别**：原地修改 vs 返回新列表
10. **reverse()**：反转列表
11. **嵌套列表**：列表中包含列表（二维列表）
12. **列表拼接**：+ 运算符拼接列表
13. **列表重复**：* 运算符重复列表
14. **in/not in**：成员判断
15. **列表拷贝陷阱**：赋值是引用，不是拷贝

### 实践技能（5个）

1. **列表增删改查**：append、insert、remove、pop、修改元素
2. **列表排序**：sort()原地排序、sorted()返回新列表
3. **列表遍历**：for循环遍历列表元素
4. **列表与字符串转换**：split()分割字符串为列表、join()拼接列表为字符串
5. **列表统计函数**：len()、max()、min()、sum()

### 重要发现

- ⚠️ **列表可变，字符串不可变**：列表可以修改元素，字符串不能
- ⚠️ **sort()修改原列表，sorted()不修改**：这是常见考点
- ⚠️ **列表赋值是引用**：list2 = list1 不是拷贝，而是引用同一个列表
- ✅ **列表切片语法与字符串完全相同**：学过字符串切片就很容易理解

## 💡 代码实践亮点

### 列表基础操作
```python
# 创建列表
fruits = ["apple", "banana", "cherry"]

# 增删改查
fruits.append("orange")      # 末尾添加
fruits.insert(1, "mango")    # 指定位置插入
fruits.remove("banana")      # 删除指定元素
fruits[0] = "grape"          # 修改元素
```

### 列表排序
```python
numbers = [3, 1, 4, 1, 5, 9, 2, 6]

# sort() - 原地修改
numbers.sort()
print(numbers)  # [1, 1, 2, 3, 4, 5, 6, 9]

# sorted() - 返回新列表
new_list = sorted(numbers)
```
