## 📖 学到的知识点

### for循环核心知识
1. **for循环的概念**：遍历可迭代对象，自动取出每个元素
2. **for循环的基本语法**：
   ```python
   for 变量 in 可迭代对象:
       代码块
   ```
3. **遍历对象**：列表、字典、元组、集合、字符串
4. **range()函数**：
   - `range(stop)`：0到stop-1
   - `range(start, stop)`：start到stop-1
   - `range(start, stop, step)`：带步长
5. **enumerate()函数**：同时获取索引和值
6. **循环嵌套**：外层每执行一次，内层执行完整一轮
7. **常见模式**：累加、计数、过滤、列表推导

### while循环核心知识
1. **while循环的概念**：条件成立时持续执行
2. **while循环的基本语法**：
   ```python
   while 条件:
       代码块
   ```
3. **break**：跳出整个循环
4. **continue**：跳过本次循环，继续下一次
5. **无限循环**：`while True:` + break控制退出
6. **while vs for**：已知次数用for，条件控制用while
7. **常见陷阱**：无限循环、循环变量忘记更新

### 综合项目：九九乘法表
1. **嵌套循环实现**：外层1-9，内层1-外层值
2. **格式化输出**：使用f-string对齐
3. **循环嵌套执行顺序**：深入理解外层与内层关系
4. **综合运用**：循环嵌套 + 条件判断 + 字符串格式化
