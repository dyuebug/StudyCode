## 📖 学到的知识点

### 函数基础核心知识
1. **函数的概念**：有名字的、可重复调用的代码块，实现代码复用和模块化
2. **def 关键字**：定义函数的语法
   ```python
   def 函数名(参数):
       函数体
       return 返回值
   ```
3. **三种参数类型**：
   - 位置参数：按顺序传入，最常用
   - 关键字参数：用参数名=值传入，顺序可变
   - 默认参数：有默认值，调用时可不传
4. **可变参数**：
   - `*args`：接收任意数量位置参数，函数内是元组
   - `**kwargs`：接收任意数量关键字参数，函数内是字典
5. **return 语句**：返回值给调用者；没有return则返回None；可返回多个值（元组）
6. **变量作用域**：
   - 局部变量：函数内部定义，只在函数内可用
   - 全局变量：函数外部定义，全局可用
   - 修改全局变量需要 `global` 声明
7. **常见陷阱**：
   - 默认参数不能用可变对象（列表、字典）
   - 忘记加括号（只引用函数，不调用）
   - return后代码不执行（死代码）

### 函数综合练习
1. **多功能计算器**：add、subtract、multiply、divide四个函数，体会单一职责
2. **学生成绩分析系统（函数版）**：

   - 函数之间相互调用，体会模块化的好处
3. **密码强度检测器**：

## 🎯 重要收获

1. **函数是代码复用的核心**：写一次，用多次
2. **模块化思维**：每个函数只做一件事（单一职责）
3. **默认参数陷阱要牢记**：不要用可变对象做默认参数
4. **return可以返回多个值**：实际是元组，用解包接收
5. **函数之间可以互相调用**：组合小函数实现复杂功能
