## 📖 学到的知识点

### 字典（Dictionary）核心知识
1. **字典的概念**：键值对（key-value）数据结构，通过键访问值
2. **字典的创建**：
   - 花括号：`{"name": "张三", "age": 18}`
   - dict()函数：`dict(name="张三", age=18)`
   - 字典推导式：`{x: x**2 for x in range(5)}`
3. **字典的访问**：
   - `[]`访问：`student["name"]`（键不存在会报错）
   - `get()`方法：`student.get("name", "默认值")`（更安全）
4. **字典的增删改查**：
   - 添加/修改：`dict[key] = value`
   - 删除：`del dict[key]`、`dict.pop(key)`
   - 查询：`key in dict`
5. **字典的遍历**：
   - `keys()`：遍历所有键
   - `values()`：遍历所有值
   - `items()`：遍历键值对
6. **字典的常用方法**：
   - `get(key, default)`：安全获取值
   - `pop(key)`：删除并返回值
   - `update(dict2)`：合并字典
   - `clear()`：清空字典
   - `setdefault(key, default)`：获取或设置默认值
7. **字典的键必须是不可变类型**：字符串、数字、元组可以，列表不行

### 元组（Tuple）核心知识
1. **元组的概念**：不可变序列，类似"只读列表"
2. **元组的创建**：
   - 圆括号：`(1, 2, 3)`
   - tuple()函数：`tuple([1, 2, 3])`
   - 单元素元组：`(1,)`（注意逗号）
3. **元组的特点**：
   - 有序：元素有固定顺序
   - 不可变：创建后不能修改
   - 支持索引和切片：与列表相同
4. **元组的解包**：`a, b = (1, 2)`（非常实用）
5. **元组的使用场景**：
   - 函数返回多个值
   - 作为字典的键（列表不行）
   - 保护数据不被修改
6. **元组 vs 列表**：
   - 元组不可变，列表可变
   - 元组更快、更省内存
   - 元组可作为字典的键

## 🎯 重要收获

1. **字典是Python最强大的数据结构**：键值对结构非常适合存储关联数据
2. **get()比[]更安全**：get()不会报KeyError，推荐使用
3. **元组的不可变性很重要**：保护数据、作为字典的键、函数返回多个值
4. **数据结构的选择**：
   - 列表：有序、可变、适合同类数据
   - 字典：键值对、适合关联数据
   - 元组：不可变、适合固定数据
5. **综合运用**：实际项目中经常需要组合使用字典、列表、元组
