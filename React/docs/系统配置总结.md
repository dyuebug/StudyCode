## ❓ 常见问题

### Q1: 提示"node: command not found"
**A**: 需要安装Node.js
```bash
# 1. 访问 https://nodejs.org/
# 2. 下载LTS版本
# 3. 安装后重启VSCode
# 4. 验证安装
node --version
```

### Q2: 找不到文件
**A**: 确保在正确的目录
```bash
# 查看当前目录
pwd

# 进入项目目录
cd E:/Code/ProjectsCode/WorkSpace/ClaudeCode/StudyCode/React

# 查看文件
ls docs/react_basics/
```

### Q3: 中文输出乱码
**A**: 设置终端编码
```bash
# Windows PowerShell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```
