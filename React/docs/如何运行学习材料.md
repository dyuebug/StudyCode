## 方式1：【推荐】使用Node.js

这是最简单、最直接的方式。

### 前置条件

确保已安装Node.js：

```bash
# 检查是否安装Node.js
node --version

# 如果显示版本号（如 v18.x.x），说明已安装
# 如果提示命令不存在，需要先安装Node.js
```

**如果未安装Node.js**：
1. 访问 https://nodejs.org/
2. 下载LTS版本（长期支持版）
3. 安装后重启VSCode

### 运行步骤

#### 步骤1：打开VSCode终端

在VSCode中按快捷键：
- Windows: `Ctrl + ~` 或 `Ctrl + J`
- Mac: `Cmd + ~` 或 `Cmd + J`

或者：菜单栏 → 终端 → 新建终端

#### 步骤2：运行JavaScript文件

在终端中输入：

```bash
# 运行第1个文件：变量声明
node docs/react_basics/01_js_variables.jsx

# 运行第2个文件：数据类型
node docs/react_basics/02_js_datatypes.jsx

# 运行第3个文件：模板字符串
node docs/react_basics/03_js_template_strings.jsx

# 运行第4个文件：综合练习
node docs/react_basics/04_practice_hello.jsx
```

#### 步骤3：查看输出

运行后，终端会显示所有的console.log输出。

### 优点

✅ 简单直接，无需额外配置
✅ 输出清晰，易于阅读
✅ 适合学习和调试

## 方式2：使用VSCode插件Code Runner

这个插件可以让你一键运行代码。

### 安装步骤

#### 步骤1：安装Code Runner插件

1. 在VSCode中按 `Ctrl + Shift + X`（Mac: `Cmd + Shift + X`）打开扩展面板
2. 搜索 "Code Runner"
3. 点击安装（作者：Jun Han）

#### 步骤2：配置Code Runner（可选）

按 `Ctrl + ,`（Mac: `Cmd + ,`）打开设置，搜索 "code runner"，勾选：
- ✅ Run In Terminal（在终端中运行）
- ✅ Clear Previous Output（清除之前的输出）

### 运行步骤

#### 方法1：使用快捷键

1. 打开要运行的.jsx文件
2. 按 `Ctrl + Alt + N`（Mac: `Cmd + Option + N`）
3. 查看终端输出

#### 方法2：使用右键菜单

1. 打开要运行的.jsx文件
2. 右键点击编辑器
3. 选择 "Run Code"
4. 查看终端输出

#### 方法3：使用播放按钮

1. 打开要运行的.jsx文件
2. 点击右上角的播放按钮 ▶️
3. 查看终端输出

### 优点

✅ 一键运行，非常方便
✅ 支持多种语言
✅ 可以快速测试代码片段

## 方式3：在浏览器中运行

如果你想在浏览器中运行（可以看到更丰富的输出）。

### 步骤1：创建HTML文件

在项目根目录创建 `test.html`：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript学习</title>
</head>
<body>
    <h1>JavaScript学习 - 打开控制台查看输出</h1>
    <p>按 F12 打开开发者工具，切换到 Console 标签页</p>

    <!-- 引入要运行的JavaScript文件 -->
    <script src="docs/react_basics/01_js_variables.jsx"></script>
</body>
</html>
```

### 步骤2：安装Live Server插件（推荐）

1. 在VSCode中按 `Ctrl + Shift + X` 打开扩展面板
2. 搜索 "Live Server"
3. 点击安装（作者：Ritwick Dey）

### 步骤3：运行

1. 右键点击 `test.html`
2. 选择 "Open with Live Server"
3. 浏览器会自动打开
4. 按 `F12` 打开开发者工具
5. 切换到 "Console" 标签页查看输出

### 切换运行不同文件

修改 `test.html` 中的 `<script>` 标签：

```html
<!-- 运行第1个文件 -->
<script src="docs/react_basics/01_js_variables.jsx"></script>

<!-- 运行第2个文件 -->
<script src="docs/react_basics/02_js_datatypes.jsx"></script>

<!-- 运行第3个文件 -->
<script src="docs/react_basics/03_js_template_strings.jsx"></script>

<!-- 运行第4个文件 -->
<script src="docs/react_basics/04_practice_hello.jsx"></script>
```

### 优点

✅ 可以看到浏览器环境的输出
✅ 支持交互式调试
✅ 更接近真实的Web开发环境

## 🎯 推荐学习流程

### 第1步：选择运行方式

**初学者推荐**：方式1（Node.js）或方式2（Code Runner）

原因：
- 简单直接
- 输出清晰
- 无需额外配置

### 第2步：按顺序学习

```bash
# 第1天学习顺序
1. node docs/react_basics/01_js_variables.jsx
2. node docs/react_basics/02_js_datatypes.jsx
3. node docs/react_basics/03_js_template_strings.jsx
4. node docs/react_basics/04_practice_hello.jsx
```

### 第3步：边学边练

1. 打开文件，阅读注释
2. 运行代码，观察输出
3. 修改代码，再次运行
4. 完成文件末尾的练习题

### 第4步：测试代码片段

如果想快速测试某段代码：

#### 方法1：使用Node.js REPL

```bash
# 在终端输入
node

# 进入交互模式，可以直接输入代码
> const name = "张三";
> console.log(`你好，${name}！`);
你好，张三！

# 按 Ctrl + C 两次退出
```

#### 方法2：创建临时文件

```bash
# 创建测试文件
echo 'console.log("Hello World");' > test.js

# 运行
node test.js

# 删除
rm test.js
```

## 常见问题

### Q1: 运行时提示 "node: command not found"

**原因**：未安装Node.js或未添加到环境变量

**解决**：
1. 访问 https://nodejs.org/ 下载安装
2. 安装后重启VSCode
3. 在终端运行 `node --version` 验证

### Q2: 运行时提示 "Cannot find module"

**原因**：文件路径不正确

**解决**：
```bash
# 确保在项目根目录
cd E:/Code/ProjectsCode/WorkSpace/ClaudeCode/StudyCode/React

# 使用正确的相对路径
node docs/react_basics/01_js_variables.jsx
```

### Q3: 输出中文乱码

**原因**：终端编码问题

**解决**：
```bash
# Windows PowerShell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8

# 或者在VSCode设置中搜索 "terminal encoding"
# 设置为 "utf8"
```

### Q4: Code Runner运行后没有输出

**原因**：未勾选 "Run In Terminal"

**解决**：
1. 按 `Ctrl + ,` 打开设置
2. 搜索 "code runner run in terminal"
3. 勾选该选项

### Q5: 想要清空终端输出

**方法**：
- Windows: 输入 `cls` 或按 `Ctrl + K`
- Mac/Linux: 输入 `clear` 或按 `Cmd + K`

## 💡 学习技巧

### 技巧1：使用VSCode的分屏功能

1. 打开.jsx文件
2. 按 `Ctrl + \` 分屏
3. 左边看代码，右边看终端输出

### 技巧2：使用代码片段快速测试

在文件末尾添加测试代码：

```javascript
// 测试区域
console.log("=".repeat(60));
console.log("测试代码");
console.log("=".repeat(60));

// 你的测试代码写在这里
const test = "Hello";
console.log(test);
```

### 技巧3：使用注释临时禁用代码

```javascript
// 单行注释：这行代码不会执行
// console.log("这行不会输出");

/*
多行注释：
这些代码都不会执行
console.log("不会输出1");
console.log("不会输出2");
*/
```

### 技巧4：使用console.log调试

```javascript
// 输出变量
const name = "张三";
console.log("name:", name);

// 输出多个变量
const age = 25;
console.log("name:", name, "age:", age);

// 输出对象
const user = { name: "张三", age: 25 };
console.log("user:", user);

// 使用console.table（更美观）
console.table(user);
```
