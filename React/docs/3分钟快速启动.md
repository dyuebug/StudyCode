# 🚀 3分钟快速启动指南

> 最简单的开始方式，3分钟内开始学习！

---

## ⚡ 超快速启动（3步）

### 第1步：打开VSCode终端

按快捷键：`Ctrl + ~` 或 `Ctrl + J`

---

### 第2步：运行脚本

**Windows用户**：
```bash
./run.bat
```

**Mac/Linux用户**：
```bash
./run.sh
```

---

### 第3步：选择文件

脚本会显示菜单，输入 `1` 开始学习第一个文件。

---

## 🎯 就这么简单！

现在你应该看到代码输出了。

---

## 📚 学习顺序

```
[1] 01. 变量声明      (30分钟)
[2] 02. 数据类型      (30分钟)
[3] 03. 模板字符串    (20分钟)
[4] 04. 综合练习      (40分钟)
```

---

## 💡 学习方法

1. **阅读输出**：仔细阅读终端中的输出
2. **打开文件**：在VSCode中打开对应的.jsx文件
3. **阅读注释**：文件中有超详细的注释
4. **修改代码**：尝试修改代码，再次运行
5. **完成练习**：完成文件末尾的练习题

---

## 🔄 重新运行

想再次运行某个文件？

```bash
# 方式1：再次运行脚本
./run.bat  # 或 ./run.sh

# 方式2：直接运行
node docs/react_basics/01_js_variables.jsx
```
