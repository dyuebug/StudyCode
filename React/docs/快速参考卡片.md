## 🎓 今日重点概念

### 变量声明
```javascript
const name = "张三";  // 常量，不能修改
let age = 25;         // 变量，可以修改
// 不要使用var
```

### 数据类型
```javascript
"文本"        // String
42           // Number
true         // Boolean
null         // Null（主动设置为空）
undefined    // Undefined（未赋值）
```

### 模板字符串
```javascript
const name = "李四";
const msg = `你好，${name}！`;  // 使用反引号和${}
```

### 类型检测
```javascript
typeof "hello"    // "string"
typeof 42         // "number"
typeof true       // "boolean"
typeof undefined  // "undefined"
typeof null       // "object" (bug!)

// 检测null的正确方式
value === null
```

## ⚠️ 常见错误

```javascript
// 错误1：使用单引号而不是反引号
'Hello, ${name}!'  // ❌ 不会替换变量
`Hello, ${name}!`  // ✅ 正确

// 错误2：const声明后尝试修改
const x = 10;
x = 20;  // ❌ 错误

// 错误3：字符串和数字相加
const age = "25";
age + 1  // "251" ❌ 字符串拼接
Number(age) + 1  // 26 ✅ 正确

// 错误4：使用==而不是===
0 == ""   // true ❌ 类型转换
0 === ""  // false ✅ 严格比较

// 错误5：使用typeof检测null
typeof null === "null"  // ❌ 错误
value === null          // ✅ 正确
```
