## ❓ 常见问题

### Q: 提示"node: command not found"
**A**: 需要安装Node.js
1. 访问 https://nodejs.org/
2. 下载LTS版本
3. 安装后重启VSCode
4. 运行 `node --version` 验证

### Q: 找不到文件
**A**: 确保在正确的目录
```bash
# 查看当前目录
pwd

# 进入项目目录
cd E:/Code/ProjectsCode/WorkSpace/ClaudeCode/StudyCode/React

# 查看文件
ls docs/react_basics/
```

### Q: 中文输出乱码
**A**: 设置终端编码
```bash
# Windows PowerShell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

### Q: VSCode会自动提示安装插件吗？
**A**: 会的！我已经配置了推荐插件列表，VSCode会自动提示安装：
- Live Server（浏览器预览）
- Code Runner（一键运行）
- ESLint（代码检查）
- Prettier（代码格式化）
