## ❓ 常见问题

### Q: 提示"node: command not found"
**A**: 需要安装Node.js
1. 访问 https://nodejs.org/
2. 下载LTS版本
3. 安装后重启VSCode

### Q: 找不到文件
**A**: 确保在正确的目录
```bash
cd E:/Code/ProjectsCode/WorkSpace/ClaudeCode/StudyCode/React
```

### Q: 中文输出乱码
**A**: 设置终端编码
```bash
# Windows PowerShell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

### Q: 如何查看文档？
